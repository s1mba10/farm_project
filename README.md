# 💊 Фарм-тренажёр

Мини-сайт для зубрёжки 1990 вопросов по фармакологии.

## Что внутри

- `data/questions.json` — все вопросы, спарсенные из исходного `.docx`
- `index.html` + `styles.css` + `app.js` — статический сайт
- `Dockerfile` + `docker-compose.yml` — деплой через nginx

## Возможности

- **Учить** — листать вопросы, нажимать ответ, видеть подсветку (зелёный/красный)
- **Тест** — отвечать без подсказок, потом проверять кнопкой
- **Ошибки** — отдельный режим: только те вопросы, что ты завалил
- **Закладки** — пометить сложные вопросы и проходить только их
- **Поиск** — по тексту вопроса и вариантов
- **Перемешивание** — вопросов и/или вариантов ответа
- **Прогресс** — сохраняется в браузере (`localStorage`)
- **Горячие клавиши**: `1-4` ответ · `←`/`→` навигация · `Enter` проверить · `B` закладка

## Запуск через Docker

```bash
docker compose up -d --build
```

Открой http://localhost:8080

Остановить:
```bash
docker compose down
```

## Запуск без Docker

Любой статический сервер. Например:

```bash
cd farm-site
python3 -m http.server 8080
```

## Перепарсинг docx

Если исходник изменился, перегенерируй `data/questions.json`:

```bash
pip install python-docx
python parse_questions.py
```
